私も、もともと通ってたDr.ストレッチを緊急事態宣言時に辞めてしまい、腰痛がどんどん悪化する一方でした。

またDr.ストレッチに行くのも良いですが、せっかくなので新しいものに挑戦してみようと、筋膜リリースの専門店を体験してみることにしました。

# 筋膜リリースとは？

筋膜とは、簡単にいうと身体を覆う膜のようなものです。

筋膜は日々の生活でどんどん筋肉にくっついてしまい、こんな問題につながっていきます。

- 筋肉の動きが悪くなる
- 血行やリンパの流れが悪くなり、むくみなどにつながる

そんな_筋肉にくっついてしまった膜を剥がすのが筋膜リリース_なわけですね。

## 筋膜リリースのやり方

歪んだ筋膜は、簡単にいうとボコボコに荒れた状態になっているそうです。

そこで、筋肉に圧力を加えることで、それを剥がすことができるとのことでした。

# 施術の流れ

施術は、基本的にこんな流れでした。

1. 電動マッサージャーをあてる
2. ローラーを転がす

これを部位ごとに行なっていきます。

もちろん筋肉によって、ローラーだけだったり、指でゴリゴリしたりと、いろいろです。

## 施術について

端的にいうと、場所によってはめちゃくちゃ痛かったです。

マッサージャーはまだ気持ちよかったのですが、ローラーは力を込めてゴリゴリ転がすのでなかなかしんどい。

そして_痛い場所は筋肉が癒着してる場所_だそうで、施術としては痛い場所ばかり攻めます。

ということで施術中はひたすら痛みに耐える感じでした。

# 施術後の変化

終わった後はまだ痛みが残ってて、整体やマッサージのような「あー軽くなった」という感じではありませんでした。

また筋肉自体をどうこうというものでもなさそうなので、「腰痛の痛みが取れました！」ともなりませんでした。

「筋肉痛になるかもよ」とのことだったので、筋肉疲労が取れるというよりはむしろ疲れが溜まる感じかもしれません。

ただ歩くと明らかに身体の可動域が広がっていて、_筋肉は重いけどなんか動かしやすい_っていう変な感じがしました。

良い整体後の、筋肉軽いし身体も動かしやすいとはまた違った感覚でした。

もしかしたら身体が動かしやすい状態って筋肉が柔らかくなってるわけじゃなくて、筋膜がリリースされた状態なのかもしれません。

となると今までの整体は、筋肉を押すことで筋膜がリリースされてたから動かしやすくなっていた？

# 感想

施術中にリラックスできるわけじゃないし、終わった後に疲労が取れるわけでもないので、疲れをとるのを目的に行くのは違うかなーという感想です。

ただ、身体の硬さを治したいと思っていたので、それを目的にするにはすごく良さそうです。

というのも、ストレッチで筋肉を伸ばすにもそもそも筋肉の可動域が狭い！という感じだったのが、筋膜リリース後はより伸びるようになったからです。

1. 筋膜リリースする
2. 筋肉が伸びるようになる
3. 筋肉が柔らかくなる

なので本質的に身体の硬さを治すには筋膜リリースを最初にするのが良いのかなーと思いました。

とはいえ日常的に通うには値がはるので、どうするかは悩みどころですねー。

もうちょっと術後の身体の調子を観察しつつ、筋膜リリースについて調べてみたいと思います！
